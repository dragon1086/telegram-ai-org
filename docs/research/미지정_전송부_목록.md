# parse_mode 미지정 전송부 목록

> parse_mode가 없는 reply_text / send_message / edit_text 호출 목록
> 이 호출들은 텍스트를 plain text로 전송 (마크다운/HTML 렌더링 없음)
> 분석 일자: 2026-03-21

## 위험도 분류 기준

- 🔴 **HIGH**: LLM 응답이나 외부 데이터를 담을 가능성 있음 → 마크다운 포함 가능
- 🟡 **MEDIUM**: 시스템 생성 메시지지만 특수문자 포함 가능
- 🟢 **LOW**: 고정 문자열 또는 이모지 전용 → plain text 전송 적절
