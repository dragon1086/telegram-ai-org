# Design Critique (디자인 리뷰 스킬)

디자인실 봇이 UI/UX 결과물을 체계적으로 검토한다.

## 검토 기준
1. **접근성**: WCAG 2.1 AA 준수 여부
2. **일관성**: 디자인 시스템과의 정합성
3. **사용성**: 직관성, 실수 방지, 회복성
4. **모바일**: 반응형 동작 확인
5. **성능**: 이미지 최적화, 렌더링 블로킹
