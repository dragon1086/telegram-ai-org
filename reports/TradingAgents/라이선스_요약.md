## Apache 2.0 핵심 조항 요약

1. **저작권 표시 유지**: 원본 라이선스 파일과 NOTICE 파일을 배포 시 포함해야 함
2. **변경사항 명시**: 원본 코드 수정 시 변경 사실을 명시해야 함
3. **특허 클레임 보호**: 기여자들의 특허를 자동으로 라이선스 부여, 특허 소송 제기 시 라이선스 자동 종료
4. **상표 사용 금지**: "TauricResearch" 상표 및 로고는 별도 허가 없이 사용 불가
