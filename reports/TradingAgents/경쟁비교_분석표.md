## 차별화 요소 분석

### TradingAgents의 독보적 강점
1. **조직 구조 시뮬레이션**: 실제 트레이딩 회사의 계층 구조를 가장 충실히 재현
2. **구조화된 디베이트**: Bullish vs Bearish 리서처 토론으로 확증 편향 방지
3. **멀티 LLM 프로바이더**: 6개 프로바이더 원스톱 지원 (경쟁사 최다)
4. **논문 검증**: arXiv 논문으로 Sharpe·수익·MDD 개선 실험적 검증
5. **Star 성장속도**: 출시 후 급격한 성장 (금융 AI 레포 중 최상위권)

### FinGPT 대비 차이점
- FinGPT: 금융 도메인 LLM 파인튜닝 중심 → TradingAgents는 일반 LLM의 에이전트 협업 중심
- TradingAgents가 최신 모델(GPT-5.x 등)을 즉시 활용 가능

### FinAgent 대비 차이점
- FinAgent: 멀티모달 데이터 + 레이어 메모리로 환각 감소 초점
- TradingAgents: 조직 시뮬레이션 + 디베이트로 의사결정 품질 초점

### OpenBB 대비 차이점
- OpenBB: 데이터 플랫폼 (분석 도구 모음)
- TradingAgents: 의사결정 프레임워크 (에이전트가 판단까지)
- 상호 보완 가능 (OpenBB 데이터 → TradingAgents 의사결정)
