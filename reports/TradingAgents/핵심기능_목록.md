## 1. 트레이딩 전략 엔진

| 기능 | 지원 여부 | 설명 |
|------|----------|------|
| 멀티 에이전트 협업 의사결정 | ✅ | 4개 애널리스트 + 2개 리서처 + 트레이더 + 리스크매니저 |
| 구조화된 디베이트 | ✅ | Bullish vs Bearish 라운드 기반 토론 |
| 5-tier 투자 등급 | ✅ (v0.2.2+) | Buy/Overweight/Hold/Underweight/Sell |
| 설정 가능한 에이전트 파라미터 | ✅ | 디베이트 라운드, 모델, 온도 조절 |
| 실시간 매매 | ❌ | 시뮬레이션 전용 (Simulated Exchange) |

## 4. 백테스트 / 시뮬레이션

| 기능 | 지원 여부 | 설명 |
|------|----------|------|
| 과거 날짜 기반 분석 | ✅ | `ta.propagate("NVDA", "2026-01-15")` |
| Simulated Exchange 실행 | ✅ | 실제 주문 없이 포트폴리오 시뮬레이션 |
| 전략 백테스트 자동화 | 부분적 | 단일 날짜 단위 실행, 기간 루프는 직접 구현 필요 |
| 성과 지표 계산 | 제한적 | 누적수익, Sharpe, MDD — 논문 기준 (자동 계산기 미내장) |

## 5. LLM 멀티프로바이더 지원

| 기능 | 설명 |
|------|------|
| Factory 패턴 | 단일 인터페이스로 6개 프로바이더 전환 |
| Deep Think / Quick Think 분리 | 역할별 최적 모델 할당 가능 |
| Anthropic Effort Level | Claude 4.5+ 추론 강도 조절 (v0.2.2+) |
| OpenAI Responses API | 최신 API 지원 (v0.2.2+) |
| 로컬 LLM (Ollama) | 프라이버시 요구 환경 지원 |
