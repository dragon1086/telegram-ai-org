## 2. 프로젝트 개요 및 목적

### 배경
TradingAgents는 UCLA 연계 연구팀(Yijia Xiao, Edward Sun, Di Luo, Wei Wang)이 2024년 12월 발표한 논문에서 시작된 프로젝트다. 핵심 아이디어는 **"단일 LLM의 한계를 넘어, 실제 트레이딩 회사가 협업하는 방식 그대로 AI 에이전트 팀을 구성하면 더 나은 투자 결정을 내릴 수 있다"**는 가설을 검증하는 것이다.

### 목적
1. 멀티에이전트 LLM의 금융 의사결정 적용 가능성 실험적 검증
2. 다양한 LLM 프로바이더를 단일 프레임워크에서 사용 가능한 연구 인프라 제공
3. 구조화된 에이전트 디베이트를 통한 확증 편향 방지

### 현재 상태 (2026-03-22)
- 최신 버전: v0.2.2 (2026년 3월)
- 연간 3회 이상 릴리즈 (활발한 유지보수)
- Trading-R1 터미널 출시 예정 (사용자 경험 대폭 개선 기대)

## 3. 아키텍처 및 에이전트 구조

### 전체 시스템 아키텍처

```
입력: ticker (종목코드) + date (분석 기준일)
         │
         ▼
┌──────────────────────────────────────────────┐
│        데이터 수집 (Alpha Vantage API)         │
│  주가 OHLCV | 재무제표 | 뉴스 | 소셜 감성      │
└──────────────────────┬───────────────────────┘
                       │
┌──────────────────────▼───────────────────────┐
│           ANALYST TEAM (4개 병렬)             │
│  [펀더멘털] [감성] [뉴스] [기술적분석]          │
└──────────────────────┬───────────────────────┘
                       │ 4개 전문 리포트
┌──────────────────────▼───────────────────────┐
│        RESEARCHER TEAM (구조화 디베이트)        │
│  [강세 리서처] ◄──N라운드──► [약세 리서처]      │
└──────────────────────┬───────────────────────┘
                       │ 균형 잡힌 시장 전망
┌──────────────────────▼───────────────────────┐
│              TRADER AGENT                    │
│    BUY / SELL / HOLD + 규모 결정              │
│    5-tier 등급 (v0.2.2+)                     │
└──────────────────────┬───────────────────────┘
                       │
┌──────────────────────▼───────────────────────┐
│          RISK MANAGEMENT + PORTFOLIO MGR     │
│    리스크 평가 → 승인/거부 → 최종 실행         │
└──────────────────────────────────────────────┘
         출력: 거래 결정 + 상세 리포트 저장
```

### 에이전트 역할 매트릭스

| 에이전트 | 팀 | 입력 | 핵심 역할 |
|---------|-----|------|----------|
| Fundamentals Analyst | Analyst | 재무제표 | 내재가치·밸류에이션 평가 |
| Sentiment Analyst | Analyst | 소셜 데이터 | 시장 심리 분석 |
| News Analyst | Analyst | 뉴스 피드 | 매크로 이벤트 분석 |
| Technical Analyst | Analyst | OHLCV | MACD·RSI 패턴 분석 |
| Bullish Researcher | Researcher | 4개 리포트 | 상승 논거 구성·방어 |
| Bearish Researcher | Researcher | 4개 리포트 | 하락 논거 구성·방어 |
| Trader Agent | Trader | 디베이트 결론 | 최종 거래 판단 |
| Risk Manager | Risk | 트레이더 결정 | 리스크 필터링 |
| Portfolio Manager | Manager | 승인 거래 | 포트폴리오 실행 |

### 오케스트레이션: LangGraph

- **방향성 그래프(DAG)** 기반 에이전트 실행
- 에이전트 간 통신: **공유 상태 객체 + 자연어 리포트**
- 메모리: **BM25** (과거 분석 리포트 키워드 검색)
